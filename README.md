# 🦟 Mata Mosquito Game

A browser-based game developed with HTML, CSS and JavaScript.

## 🌐 Live Demo

👉 https://alexandrenino.github.io/mata-mosquito-game/

---

## 🎮 Features

- Three difficulty levels
- Random mosquito size
- Random mosquito direction
- Random mosquito position
- Life system
- Countdown timer
- Victory and Game Over screens
- Responsive gameplay

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 4

---

## 🚀 How to Play

1. Choose a difficulty level.
2. Click **Start Game**.
3. Click on every mosquito before it disappears.
4. Lose three lives and it's Game Over.
5. Survive until the timer reaches zero to win.
